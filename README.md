# MachineHub Demo — React Vite App

## Quick Start
```bash
npm install
npm run dev
```
Then open http://localhost:5173

## Demo Credentials
- **Admin Login:** admin@machinehub.in / Admin@123
- **User:** Register a new account (OTP shown as toast notification in demo)

## Features
- 🏠 Premium Landing Page with animations & counter stats
- 🏭 Machines & Accessories Catalog (10 products, filter/sort/search)
- 📋 Query/Contact Form
- 🔐 User Register (OTP verification) + Login
- 🛒 Request to Buy modal (login required)
- 📊 Admin Dashboard (requests management, product CRUD)
- 📧 Simulated email notification on buy request (console.log)

## Pages
| Page | Route |
|------|-------|
| Landing | Home |
| Catalog | Machines & Accessories |
| Query | Contact Form |
| Login | User Authentication |
| Register | OTP Verification Flow |
| Admin | Dashboard (admin only) |

## Tech Stack
- React 18 + Vite 5
- Pure CSS-in-JS (no extra libraries)
- localStorage for user accounts (demo only)
- Bebas Neue + DM Sans fonts (Google Fonts)
